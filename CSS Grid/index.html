Can either use first method, where you define the amount of columns/rows and select each div using 'grid-column-start'. Here you would start from the lines - so a grid with 2 columns has 3 lines.
To define it, use 'grid-column-start' and 'grid-column-end'. Or, can simplify to just 'grid-column 1/3.
Can do the exact same for rows, using 'grid-row'.
To simplify code, can use 'grid-template-columns: repeat(3, 20%)'. You can write 'grid-template-columns: repeat(autofit (or-autofill), 400px) to make it responsive (like flex-wrap)'
Sometimes you must combine both columns and rows as the page will automatically lay them out in the wrong way.
Can add 'grid-column-gap' and to rows to space things out.

The other (easier) method is using 'grid-area: name' where the name can be anything you want.
Here, once you have named each box with 'grid-area', you show it in the section itself, using 'grid-template-areas'.
You define each row by using quotes " " and write each name of the box for where it needs to be. To stretch a box, just write its name multiple times. For an empty space, just use a fullstop . Can use 'justify-items' and 'align-items' when in the section to show where the item will be in its grid.
To effect just one specific grid, use 'align-self' for that specific division.
Can define each grid as a flex-container, and go on to edit them from there.
